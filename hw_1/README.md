Що таке Node.js. Початок роботи
Node.js представляє середовище виконання коду на JavaScript, яке побудоване на основі рушія JavaScript Chrome V8, що дозволяє транслювати виклики на мові JavaScript у машинний код. Node.js передусім призначений для створення серверних додатків на мові JavaScript. Хоча також існують проекти для написання десктопних додатків (Electron) і навіть для створення коду для мікроконтролерів. Але передусім ми говоримо про Node.js як про платформу для створення веб-додатків.

Node.js є відкритим проектом, вихідний код якого можна переглянути на github.com.

Встановлення Node.js

Після успішної установки ви можете ввести в командному рядку/терміналі команду node -v, і нам відобразиться поточна версія Node.js:

Інструменти розробки
Для розробки під Node.js достатньо найпростіше текстового редактора, зокрема, Notepad++. Також можна використовувати більш просунуті редактори типу Atom, Sublime, Visual Studio Code, або середовища розробки, які підтримують роботу з Node.js, наприклад, Visual Studio або WebStorm.

REPL
Після встановлення Node.js нам стає доступним такий інструмент як REPL. REPL (Read-Eval-Print Loop) представляє можливість запуску виразів на мові JavaScript у командному рядку або терміналі.

Так, запустимо командний рядок (на Windows) або термінал (на OS X або Linux) і введемо команду node. Після введення цієї команди ми можемо виконувати різні вирази на JavaScript:

Copy code
C:\WINDOWS\system32>node
> 2+6
8
>
Або використовувати якусь функцію JS:

Copy code
> console.log("Hello NodeJS");
Hello NodeJS
undefined
>
Можна визначати свої функції і потім їх викликати, наприклад, піднесення числа до квадрату:

Copy code
> function sqrt(x){return x * x;}
undefined
> sqrt(5)
25
>

Виконання файлу
Замість того, щоб вводити весь код безпосередньо в консоль, зручніше винести його у зовнішній файл. Наприклад, створимо на жорсткому диску новий каталог, припустимо, C:\node\helloapp, у який помістимо новий файл app.js з таким кодом:

Copy code
console.log('Hello world');
У командному рядку перейдемо за допомогою команди cd до каталогу helloapp, а потім виконаємо команду:

Copy code
node app.js
Це виведе в консоль:

Copy code
Hello world
