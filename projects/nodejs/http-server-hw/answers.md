Homework 7 "Власний HTTP-сервер на Node.js"

Відповіді на запитання:

1.Чому при відкритті http://localhost:3000 не потрібен DNS? (Підказка: що таке localhost і яка його IP-адреса?)

Тому що Localhost є доменним ім'ям локального комп’ютера і при відкритті http://localhost:3000 комп’ютер впізнає свою IP-адресу.


2.Що станеться, якщо запустити два сервери на одному порту 3000? Спробуйте запустити node server.js у другому терміналі, поки перший працює, і опишіть текст помилки.

В тексті помилки вказано, що порт 3000 вже використовується і сервер не запустився.


3.Для чого потрібен заголовок Content-Type? Що зміниться, якщо його прибрати?

Content-Type вказує формат даних. Якщо його прибрати, то браузер може некоректно відтворити код.


4.У якому місці коду визначається, який код стану поверне сервер?

У цих місцях:

res.writeHead(200, { 'Content-Type': 'text/plain; charset=utf-8' }

res.writeHead(500, { 'Content-Type': 'text/plain; charset=utf-8' }

res.writeHead(404, { 'Content-Type': 'text/plain; charset=utf-8' }
